# Day 02 - Socket Communication

This day explores the socket lifecycle, TCP communication patterns, connection management, timeout handling, and multi-client concurrency with production-quality, educational labs.

Learning objectives
- Understand socket lifecycle: CREATE → CONNECT → COMMUNICATE → CLOSE
- Build and run TCP echo services and multi-client servers
- Observe timeout handling and resource cleanup
- Capture and analyze traffic with Wireshark
- Prepare for interview questions on sockets and concurrency

- `Exercise-02-Multi-Client-Echo-Server` (port 9003)
