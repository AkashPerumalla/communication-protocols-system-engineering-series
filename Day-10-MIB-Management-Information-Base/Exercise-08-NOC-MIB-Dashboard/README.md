# Exercise 08 - NOC MIB Dashboard

## Objective
Aggregate device health into a centralized NOC-style dashboard.

## What You Learn
- Centralized monitoring
- Status aggregation
- Operational visibility

## Run
```bash
java -cp out/day10 NocMibDashboard
```

## Expected Output
- `NOC DASHBOARD`
- ONLINE/OFFLINE status rows
- CPU, memory, temperature, and alarm state columns
