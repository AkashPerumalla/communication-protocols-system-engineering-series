# Exercise 08 - NOC Control Dashboard

## Objective
Display inventory, telemetry, alarms, recent commands, and recovery actions in one operational view.

## Marker
NOC CONTROL DASHBOARD

## Steps
1. Call `GET /api/dashboard`.
2. Review the device inventory, telemetry snapshot, alarms, commands, and recovery actions.
3. Correlate telemetry to command outcomes.

## Expected Output
- The dashboard aggregates the control plane view.
- Recovery actions are easy to trace.
- The response includes `NOC CONTROL DASHBOARD`.
