# Exercise 08 - NOC Dashboard

This exercise prints a console-based NOC summary with periodic refresh cycles.

## Learn
- Dashboard aggregation
- Alarm counts
- Fleet visibility

## Run
```bash
java -cp out/day09 NocDashboardSimulator
```

## Expected Output
- Devices online
- Devices offline
- Critical alarms
- Major alarms
- Average CPU
- Average memory
