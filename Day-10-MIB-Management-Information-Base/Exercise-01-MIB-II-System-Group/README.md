# Exercise 01 - MIB-II System Group

## Objective
Explore the standard MIB-II system group and its scalar objects.

## What You Learn
- MIB-II fundamentals
- Scalar objects
- sysDescr, sysObjectID, sysUpTime, sysContact, sysName, sysLocation, and sysServices

## Run
```bash
java -cp out/day10 SystemGroupExplorer
```

## Expected Output
- `sysDescr`
- `sysName`
- `sysLocation`
- A readable list of MIB-II system objects
