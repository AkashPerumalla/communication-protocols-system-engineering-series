# Day 01 Runner Scripts

These helper scripts launch each exercise with a single command.

## macOS / Linux

- `run-tcp-server.sh`
- `run-tcp-client.sh`
- `run-udp-server.sh`
- `run-udp-client.sh`
- `run-tcp-benchmark.sh`
- `run-udp-benchmark.sh`

## Windows

- `run-tcp-server.bat`
- `run-tcp-client.bat`
- `run-udp-server.bat`
- `run-udp-client.bat`
- `run-tcp-benchmark.bat`
- `run-udp-benchmark.bat`

## Notes

- Start the server script first for each client/server lab.
- The benchmark scripts expect the TCP and UDP servers to already be running.
- Run the scripts from a terminal so you can keep the server process visible.
