# Screenshots

This folder contains Serial Monitor screenshots and serial communication logs from the UART exercises.

Use these as reference for expected outputs when running the sketches in Arduino IDE.

## How to Capture

1. Open Arduino IDE with the exercise sketch loaded
2. Connect the Arduino or ESP32 board via USB
3. Open the Serial Monitor: Tools → Serial Monitor
4. Select the correct baud rate (9600 for Ex 1-2, 115200 for Ex 3-5)
5. Screenshot the Serial Monitor output
6. Save as PNG in this folder with descriptive name
