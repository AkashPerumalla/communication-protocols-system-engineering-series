# Day 01 - TCP vs UDP

## Objective

Understand the differences between TCP and UDP by implementing client-server communication using Java Socket Programming.

## Learning Outcomes

- Connection-oriented communication
- Connectionless communication
- Reliability vs Speed
- Packet ordering
- Error handling

## Exercises

### Exercise 1
TCP Client-Server Communication

### Exercise 2
UDP Client-Server Communication

### Exercise 3
Compare TCP and UDP using Wireshark

## TCP Characteristics

- Reliable
- Connection Oriented
- Ordered Delivery
- Error Checking

## UDP Characteristics

- Fast
- Connectionless
- No Delivery Guarantee
- Lower Overhead

## Real World Usage

### TCP

- HTTP/HTTPS
- SSH
- FTP
- Email

### UDP

- DNS
- VoIP
- Video Streaming
- Online Gaming
