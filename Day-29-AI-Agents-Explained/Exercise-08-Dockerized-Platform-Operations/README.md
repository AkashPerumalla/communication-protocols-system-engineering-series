# Exercise 08 - Dockerized Platform Operations

## Objective

Run the Day-29 platform in Docker and validate health and agent workflow endpoints.

## Steps

1. Build and start containers with `docker compose up --build`.
2. Validate `GET /health` and `POST /agent/chat`.
3. Optionally enable PostgreSQL profile.

## Solution

Run `run.sh` to execute container-level validation commands.

## Outcome

You confirm the platform is portable and deployment-ready.
