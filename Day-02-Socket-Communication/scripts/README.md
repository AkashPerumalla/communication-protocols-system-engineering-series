# Day 02 Runner Scripts

Use the following scripts on macOS/Linux (make them executable with `chmod +x`):

- `run-echo-server.sh` / `run-echo-client.sh`
- `run-multi-server.sh` / `run-multi-client.sh`
- `run-lifecycle-server.sh` / `run-lifecycle-client.sh`
- `run-timeout-server.sh` / `run-timeout-client.sh`

Windows `.bat` variants can be added similarly if needed.
